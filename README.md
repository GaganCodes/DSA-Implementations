# DSA-Implementations

Implementation of common Data Structures and Algorithms in Python3.

Data Structures implemented:
1. Arrays/Lists
2. Linked List
3. Stack
4. Queue
5. Hash Set
6. Hash Map
7. Adjaceny List (graphs)
8. Heap
9. Binary Search Tree

Algorithms implemented:
1. Merge Sort
2. Quick Sort
3. Selection Sort
4. Heap Sort
5. Breadth First Search
6. Depth First Search
